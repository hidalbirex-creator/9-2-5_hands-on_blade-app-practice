# blade-app-practice

## 概要
COACHTECH 教材 Tutorial 9-2「Bladeテンプレート ハンズオン演習」で作成した成果物です。
フロントから提供されたBladeファイル（products/index.blade.php）を正しく読み解き、
コントローラーとルーティングを設定して、ブラウザで商品一覧が正しく表示されるようにする。

## 使用技術
- PHP 8.x
- Laravel 10.x
- Blade テンプレート

## 学んだこと
- (object)[]で連想配列はオブジェクトになる。連想配列のままでエラーになった。
-　now()はエクセルだけかと思ったが機能した
  
## 動作確認
http://localhost/products にアクセス
